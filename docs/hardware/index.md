# Hardware Setup

This section explains how to properly connect and prepare VNM sim racing hardware before configuring the devices in VNM SimCenter.

Before launching the software, make sure all devices are correctly connected and powered.

Proper hardware setup ensures stable communication between the devices and the computer and prevents potential configuration issues.

## Setup Checklist

- [ ] Read the [Safety Notice](safety.md) before proceeding
- [ ] Connect all devices via [USB](usb-connection.md)
- [ ] Connect [power supplies](power-connection.md) where required
- [ ] [Mount the wheelbase](mounting.md) securely
- [ ] [Attach the steering wheel](steering-wheel.md)
- [ ] Connect [additional devices](additional-devices.md)
- [ ] Follow the [Power-On Procedure](power-on.md)
- [ ] [Verify device detection](device-detection.md) in VNM SimCenter
